Le Stampe

Le stampe si basano sulla costruzione di una struttura che mescola elementi HTML con elementi definiti a livello di Genropy.
Gli elementi di Genropy sono:
  • Layout
  • Row
  • Cell
Esempio:
#import della classe GnrHtmlBuilder​
from gnr.core.gnrhtml import GnrHtmlBuilder
#istanzio un builder​
builder = GnrHtmlBuilder()
#inizializzo sorgente passando gli attributi al body
builder.initializeSrc(body_attributes=dict(background='white',height='297mm',width='210mm'))
#creo il layout radice
l = builder.body.layout(height=290,width=200,top=5,left=5)
#definisco la riga di top specificando l'altezza
top = l.row(height=50)
#definisco la cella di top_left specificando la larghezza
top_left = top.cell(width=100)
#definisco la cella di top_center che non avendo larghezza è elastica e occupa tutto lo spazio disponibile
top_center = top.cell()
top_right = top.cell(width=40)
#definisco la riga di center che non avendo altezza è elastica e occupa tutto lo spazio disponibile
center = l.row()
bottom = l.row(height=20)
bottom_left = bottom.cell(width=100)
bottom_center = bottom.cell()
bottom_right = bottom.cell(width=40)
#mettiamo un elemento html nella cella top_center
top_center.div('Hello world',color='red',font_size='80px',text_align='center')
